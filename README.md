# Gamdl GUI

A cross-platform GUI application for managing and using `gamdl`.

## Features
- Install `gamdl` automatically.
- Check for updates and notify users.
- Cross-platform compatibility: Windows, MacOS, Linux.

## Installation
1. Download the latest release for your platform.
2. Run the executable file to start the app.
